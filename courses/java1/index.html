Giới thiệu khóa học

 

Chủ đề khóa học:  Phát triển ứng dụng Java  cấp độ 1

 

Thời lượng: 80 giờ

 

Đối tượng học viên

Khóa học này dành các sinh viên, học sinh, yêu thích lập trình muốn làm quen với ngôn ngữ lập trình Java.

Sinh viên đại học, Lập trình viên ít kinh nghiệm cũng có thể theo học.

Học viên không cần có kiến thức về lập trình.

Mô tả học viên

·         Có laptop.

·         Yêu thích hoặc muốn trải nghiệm về lập trỉnh.

Mục tiêu khóa học

Mục tiêu chung của khóa học là giúp các bạn làm quen với Java hoặc ôn lại kiến thức Java, trải nghiệm cách làm các phần mềm Java đơn giản, rèn luyện dần các thói quen và kỹ năng tốt trong lập trình, khai phá tiềm năng lập trình.

 

Phương pháp đào tạo

Giảng viên kiêm 3 vai trò: Giảng dạy (Teaching), Huấn luận (Training), Cố vấn (Mentoring) nhằm giúp học viên nắm được kiến thức, có kỹ năng và nhận được lời tư vấn phù hợp.

2.9 triệu / khóa (2 tháng).

Nội dung

·         Ôn tập về nguyên tắc vận hành của hệ điều hành, nguyên tắc hoạt động của chương trình máy tính.

·         Trải nghiệm và giải thích chương trình đơn giản trên hệ điều hành Windows

·         Ôn tập về cách thức hoạt động của phần mềm trên nền tảng mạng Internet.

·         Trải nghiệm và giải thích chương trình chạy tren nền tảng mạng Internet.

·         Làm quen với công nghệ lập trình, ngôn ngữ lập trình Java.

·         Phát triển ứng dụng trên Windows với thư viện SWT (của IBM)

·         Làm quen với kiến trúc ứng dụng trên nền tảng Internet.

·         Phát triển ứng dụng trên nền tảng Internet với công nghệ Java, Servlet; MySQL

·         [Nâng cao  dành cho học viên có năng lực vượt trội]:

o   Phát triển ứng dụng với Bootstrap, Spring, MySQL

o   Kiểm soát chất lượng mã nguồn bằng Unit Testing

Nội dung khóa Phát triển ứng dụng Java  cấp độ 2

·         Trải nghiệm và khai thác thư viện lập trình JDK.

·         Ôn tập và trải nghiệm phương pháp lập trình hướng đối tượng (OOP)

·         Nâng cao kỹ năng lập trình với ứng dụng trên Windows với thư viện SWT (của IBM)

·         Nâng cao kỹ năng lập trình với ứng dụng trên nền tảng Internet với công nghệ Java, Servlet; MySQL.

·         Phát triển ứng dụng Internet với Springboot, Boostrap, Thymeleaf.

·         Làm quen với lập trình API Restful, micro services

·         [Nâng cao  dành cho học viên có năng lực vượt trội]:

o   Phát triển ứng dụng với mô hình MVC

o   Kiểm soát chất lượng mã nguồn bằng Checkstyle, Unit Testing
