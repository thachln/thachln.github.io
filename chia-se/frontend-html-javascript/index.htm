Bài 1 – Làm quen HTML

HTML là gì?

Bạn hãy giải trí một chút nếu đang dùng Windows. Thử mở cửa sổ lệnh và gõ lệnh sau:

set /?

Màn hình sẽ hiển thị ra thông báo hướng dẫn như sau:

Như vậy là lệnh set với tham số /? thì lệnh sẽ hiển thị ra màn hình nội dung hướng dẫn cách sử dụng lệnh.

Viết ứng dụng dạng console

Câu hỏi đặt ra cho bạn là bạn có thể viết một lệnh tương tự để người dùng máy tính của bạn khi gõ vào thì hiển thị ra thông tin gì đó không? Ví dụ khi gõ lệnh hello thì máy tính hiển thị ra câu thông báo như sau:

Xin chào bạn!

Hãy gõ thử lệnh hello vào cửa sổ lệnh của Windows, có phải máy tính bạn hiển thị ra thông báo bên dưới không?

Phần giải thích ý nghĩa câu thông báo lỗi như bên dưới xin nhường lại cho bạn:

… is not recognized as an internal or external command,

operable program or batch file.

Chúng ta bắt tay vào “lập trình” ngay lệnh hello nhé.

Bước 1: Tạo file .cmd

Tạo file hello.cmd trong thư mục D:\RunNow với nội dung như sau:

@ECHO OFF

ECHO Xin chao ban!

Bước 2: Thiết lập biến môi trường PATH

Thiết lập biến môi trường PATH chứa đường dẫn D:\RunNow.

Bước 3: Thử chạy lệnh hello trong cửa sổ lệnh

Viết ứng dụng dạng web

Câu hỏi đặt ra tiếp theo cho bạn là làm sao “phần mềm” hello bạn viết ở trên người khác có thể dùng được mà không phải ngồi trên máy tính của bạn để gõ lệnh hello?

Tạo file D:\MyHTML5\hello\hello.html có nội dung sau:

<html>

    <body>

        Hello World!

    </body>

</html>

Dùng phần mềm Browser (gọi là trình duyệt) ví dụ như: Microsoft Edge, Google Chrome hoặc Mozilla Firefore mở file hello.html này lệnh sẽ có giao diện như sau:

Đây là file trên máy bạn (nhìn nội dung trên thanh Address là: file:///D:/MyHTML5/hello/hello.html).

Câu hỏi đặt ra là làm sao “phần mềm” Hello World này có thể xuất hiện được trên máy người khác mà không cần phải chép file hello.html đưa cho họ?

Ý tưởng như sơ đồ sau:

Bạn cần một phần mềm gọi là Web Server (trong hình ở trên tôi dùng Nginx - http://nginx.org) để phục vụ cho người dùng. Phía người dùng sẽ sử dụng một phần mềm Browser để nhập địa chỉ http://<IP Address>/hello.html để truy cập được “phần mềm” Hello World của bạn.

Chuẩn bị Nginx

Tải Nginx cho Windows bản ổn định tại thời điểm bài này được viết tại:

http://nginx.org/download/nginx-1.20.1.zip

Giải nén ra thư mục D:\RunNow có kết quả như sau:

Tiếp theo bạn nên tạo 2 script để khởi động và tắt nginx như sau:

Tạo script start.cmd

nginx.exe

Tạo script stop.cmd

nginx.exe -s stop

Khởi động Nginx

Khởi động nginx bằng cách thực thi script start.cmd. Kết quả là cửa sổ hiện ra như bên dưới. Chú ý đừng bấm chuột vào cửa sổ đó. Nếu lỡ bấm thì nhấn Enter.

Mở trình duyệt truy cập địa chỉ http://localhost sẽ ra kết quả nhu sau:

Bạn có thể lấy địa chỉ mạng (IP Address) của máy bạn (giả định đang dùng Wifi) bằng lệnh:

ipconfig

Tìm đến mục “Wireless LAN adapter Wi-Fi” > IPv4 Address. Ví dụ hình bên dưới IP Address của tôi là 192.168.1.4

Lúc này bạn có thể cung cấp cho những người đang dùng cùng Wifi với bạn có thể truy cập địa chỉ http://192.168.1.4

Mở rộng thêm nếu bạn mua một tên miền và biết cách cấu hình cái cục phát Wifi thì có thể cấu hình cho cả thế giới sử dụng được phần mềm “Hello World” của bạn. Ví dụ tôi cấu hình trang https://myworkspace.vn trỏ và cái laptop đang chạy nginx đã hướng dẫn ở trên.

Cấu hình Nginx cho ứng dụng

Thêm đoạn cấu hình sau

            alias D:/MyHTML5/hello;

            autoindex on;

        } 

 

 vào file:

D:\RunNow\nginx-1.20.1\conf\nginx.conf

Kết quả:

Tắt và chạy lại nginx bằng 2 script đã viết start.cmd và stop.cmd

Lúc này bạn vào địa chỉ http://localhost/myweb thì kết quả như sau:

Bấm vào link hello.html thì sẽ ra kết quả “Hello World” mà bạn đã làm.

Bài 2 – Làm quen Javascript

Dùng javascript

Copy file hello.html thành hello-javascript.html và thêm các nội dung mới như sau:

<html>

    <head>

        <script src="https://code.jquery.com/jquery-1.9.1.min.js"></script>

    </head>

    <body>

        Hello World!

 

        <script>

            $(document).ready(function() {

                console.log("Document loaded");

            });

        </script>

    </body>

</html>

Giải thích:

·        Đoạn bôi vàng đâu tiên để khai báo sử dụng thư viện jQuery tại địa chỉ trong thẻ script, thuộc tính src.

·        Đoạn bôi vàng thứ hai sử dụng cú pháp của thư viện jQuery

Thử mở địa chỉ http://localhost/myweb/hello-javascript.html và dùng chức năng Inspect của trình duyệt (nhấn phím tắt Ctrl + Shift + I) để xem thẻ Console khi màn hình được hiển thị đầy đủ trên máy người dùng.

Nâng cấp

Bổ sung file html vài chỗ màu vàng như sau:

<html>

    <head>

        <meta charset="UTF-8">

        <script src="https://code.jquery.com/jquery-1.9.1.min.js"></script>

    </head>

    <body>

        <p id='message'>Hello World!</p>

 

        <script>

            $( document ).ready(function() {

                $("#message").html("Chào bạn!");

                console.log("Document loaded");

            });

        </script>

    </body>

</html>

Hãy tự mở lại trang web và đọc tài liệu sau để tự giải thích nhé:

·        https://www.w3schools.com/tags/att_meta_charset.asp

·        https://www.w3schools.com/tags/att_id.asp

·        https://www.w3schools.com/jquery/jquery_dom_set.asp

Đúc kết

·        Nhu vậy bạn đã biết là có một thư viện jQuery giúp bạn làm màn hình ứng dụng Web. Trong đó bạn có thể lập trình Javascript mà nó sẽ được thực thi trên máy (cụ thể là trên phần mềm Browser) của người dùng.
